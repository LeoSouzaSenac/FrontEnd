# Trabalho Prático - Criação de um Website do Zero com HTML e CSS

## Objetivo
Neste trabalho, os grupos devem escolher um tema de sua preferência e criar um website do zero utilizando **HTML** e **CSS**. 
O objetivo é aplicar os conhecimentos adquiridos em sala de aula, utilizando as tags corretas e aplicando **Flexbox** para 
o layout do site.

## Requisitos

### Estrutura do site
O site deverá conter as seguintes seções obrigatórias:

1. **Cabeçalho (Header)**:
   - O cabeçalho deverá conter o **nome do site** e um **menu de navegação** com links para pelo menos **três páginas diferentes**.
   - Utilize a tag `<header>` para definir o cabeçalho e a tag `<nav>` para o menu de navegação.
   - O menu deve estar estilizado de forma responsiva usando **Flexbox**.

2. **Banner**:
   - Logo abaixo do cabeçalho, adicione um banner com uma imagem e também texto que represente o tema do site.
   - Utilize a tag `<section>` ou `<div>` para definir essa área.

3. **Conteúdo principal**:
   - O site deve ter uma seção de **conteúdo principal**, onde serão apresentados os textos e imagens relacionados ao tema escolhido.
   - Use a tag `<main>` para o conteúdo principal.

4. **Rodapé (Footer)**:
   - O rodapé deve conter informações como **créditos**, **redes sociais**, ou qualquer outro conteúdo relevante.
   - Utilize a tag `<footer>`.

### Requisitos de Navegação
- O site deverá ter pelo menos **3 páginas HTML interligadas**:
   - Exemplo: página inicial, sobre, e contato.
   - Use links (`<a href="pagina.html">`) no menu para navegar entre essas páginas.

### Requisitos de Estilo (CSS)
- Use **Flexbox** para organizar o layout das seções (menu, banner, conteúdo, rodapé).
- Estilize o site utilizando um **arquivo CSS externo** (`style.css`).
- O design deve ser **responsivo**, ou seja, deve se adaptar bem tanto em telas de desktop quanto em dispositivos móveis.
- Os estilos devem incluir:
  - Cores e fontes condizentes com o tema escolhido.
  - Espaçamentos adequados para manter o conteúdo bem organizado.

### Tags e Elementos Importantes
Os alunos deverão usar as tags corretas para cada parte do site:
- `<header>`, `<nav>`, `<section>`, `<footer>`, `<main>`, `<div>`, `<img>`, `<a>`, etc.
- Utilize **Flexbox** para posicionamento e organização.

### Outros Requisitos
- O site deve estar devidamente comentado. Comentários no código (`<!-- comentário -->`) devem explicar a estrutura e funcionalidade de partes importantes do HTML e CSS.
- Todos os arquivos (HTML e CSS) devem ser organizados em uma pasta chamada `site_tema`.
- Não esqueça do **FAVICOM** E DO **TÍTULO** do seu site!

## Critérios de Avaliação
1. **Estrutura do HTML**: uso correto das tags e semântica
2. **Navegação**: links entre as páginas funcionando corretamente
3. **Layout e Flexbox**: uso eficiente do Flexbox para organizar o layout
4. **Design e Responsividade**: design adequado ao tema e adaptação em diferentes dispositivos
5. **Comentários e Organização do Código**: clareza e comentários no código
6. **Criatividade e Inovação**: originalidade e inovação no design
